# SauceDemo Bug Report
l
## Bug ID
BUG-001
## Title
Shopping cart badge does not update immediately after removing an item.
## Environment
- Website: https://www.saucedemo.com/
- Browser: Chrome Mobile
- Device: Android
- OS: Android
## Severity
Medium
## Priority
Medium
## Preconditions
- User is logged in.
- At least one product has been added to the cart.
## Steps to Reproduce
1. Log in with valid credentials.
2. Add a product to the cart.
3. Open the cart.
4. Remove the product.
5. Observe the cart badge.
## Expected Result
The cart badge should immediately update to show the correct number of items.
## Actual Result
The cart badge does not update immediately until the page is refreshed.
## Status
Open
